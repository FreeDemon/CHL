What is C HyperText Library really?

CHL, short for C Hypertext Library, is a library written in C to be used to write web applications. If you have ever used a language designed for creating server-side webscripts, PHP or ASP for instance, you will feel quite familiar with CHL, since CHL is used for the exact same purpose. The only difference, however, is that CHL provides you with the ability to write those Web applications in pure C, instead of having to learn a new language.

Many people, including myself, also finds great joy in writing code in C, so why not do it for developing web applications as well?

Core Features

Managing Cookies

A carefully crafted set of instructions for creation, deletion and manipulation of HTTP cookies.

Handling of HTTP Data

An easy to use, dedicated API for parsing and fetching HTTP data, such as POST and GET. As well as instructions for full manipulation of HTTP headers.

Various Plugins

Comes with various hand-crafted plugins for your pleasure only, thanks to CHL's Open-Source code. See Github.

Inline Functions

Ships with the very unique feature of supporting inline functions inside the HTML files, much like higher-level languages like PHP, but here using the "<{ func(); }>" syntax.
